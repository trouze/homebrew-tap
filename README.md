# homebrew-tap

Homebrew tap for [trouze](https://github.com/trouze)'s tools.

## Install

```bash
brew tap trouze/tap
```

## Formulae

### rocketlane-cli

CLI tool for [Rocketlane](https://rocketlane.com) — Professional Services Automation from the terminal.

```bash
brew install rocketlane-cli
```

See the [rocketlane-cli repo](https://github.com/trouze/rocketlane-cli) for full usage docs.
